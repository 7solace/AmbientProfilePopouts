# AmbientProfilePopouts

A customizable BetterDiscord plugin focused on immersive visuals, smooth UI animations, and profile enhancements.

---

## Installation

1. Download `AmbientProfilePopouts.plugin.js`
2. Place the file inside your BetterDiscord plugins folder
3. Enable the plugin from BetterDiscord settings


---
## Features

### Ambient & Visual Effects
- Adaptive ambient glow effects for profile popouts and modals  
- Modern glassmorphism UI with blur/backdrop filters
- Customizable blur, transparency, sheen intensity, and edge lighting
- Smooth visual integration with Discord's interface

### Animation System
- Per-area animations for:
  - Channel switching
  - Server switching
  - Sidebars
  - Popouts & modals
  - Emoji picker
  - Context menus
  - Message inputs...

### Animation Types
- Fade
- Slide
- Scale
- Blur
- Flip and more.

### Productivity & Utility
- Hide typing indicators
- Safer link checking system
- Shift + Click message copying

### Customization
- Built-in settings panel
- Adjustable animation speeds
- Visual tuning sliders
- Reset to default settings

### Technical Features
- Dynamic DOM integration using MutationObserver
- Lightweight and responsive behavior
- Automatic update checking system
